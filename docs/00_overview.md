# 유니콘클래스 프로젝트 개요

## 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 유니콘클래스 (Unicorn Class) |
| 버전 | v3.0 |
| 유형 | 온라인 강의 플랫폼 (VOD + 무료 웨비나) |
| 타깃 | N잡, 커머스, AI, SNS 마케팅, 유튜브, 부업 분야 수강생 |
| 주요 지표 | 수강생 24만+, 수료 4만+, 운영 13년+, 파트너 강사 50+ |

---

## 기술 스택

| 레이어 | 기술 |
|--------|------|
| 언어 | PHP 8.2 |
| 프레임워크 | Vanilla PHP (PSR-4 Composer 자동로딩) |
| 데이터베이스 | MariaDB 10.11 |
| 웹서버 | Apache 2.4 (mod_rewrite, PHP-FPM) |
| 컨테이너 | Docker / Docker Compose |
| 영상 플레이어 | Vimeo Embed (iframe) |
| 결제 | PG사 연동 (신용카드, 체크카드, 계좌이체) |
| 소셜 로그인 | 카카오, 네이버 OAuth 2.0 |
| 소통 | 카카오 오픈채팅 (강의별 링크) |
| 관리 도구 | phpMyAdmin 5.x |

---

## 아키텍처 구조

```
public/           ← DocumentRoot (Apache), 진입점 index.php
src/
  Controller/     ← 얇은 컨트롤러, 요청 수신·응답만
  Repository/     ← 모든 SQL 처리 (PDO prepared statements)
  Service/        ← 비즈니스 로직
  Model/          ← VO/DTO
  Core/           ← Router, DB, Session, CSRF, Auth
database/         ← 마이그레이션 SQL 파일
docs/             ← 프로젝트 문서
docker/           ← Dockerfile, 설정 파일
```

## 주요 기능 목록

### 1. 회원
- 이메일/아이디 회원가입 (약관 동의, 중복 체크)
- 이메일·소셜 로그인 (카카오, 네이버 OAuth)
- 아이디 찾기 (SMS 인증), 비밀번호 재설정 (이메일 인증)
- 마이페이지 (정보수정, 회원탈퇴)

### 2. 강의 (클래스)
- 강의 목록: 카테고리 필터 (전체/AI/SNS/디자인·영상/마케팅/유튜브/커머스), 무료/프리미엄 탭
- 강의 상세: 설명 탭, Q&A, 후기, 수강신청/결제
- VOD 플레이어: Vimeo Embed, 챕터 목록, 강의자료 다운로드, 오픈채팅 링크
- 수강률 추적 (`lc_progress`), 수료증 발급 (80% 이상)
- 찜 기능, 장바구니, 좋아요/공유

### 3. 결제
- 강의 구매 (신용카드, 체크카드, 계좌이체)
- 결제 내역 조회, 영수증 발급
- 환불 신청 (7일 이내·수강률 33% 미만 전액, 이후 잔여 비례)
- 수강 기간 만료 시 기간 연장 재구매

### 4. 강사
- 강사 목록/상세 (프로필, 경력, 소셜 링크, 강의 포트폴리오)
- 강사 지원 폼 (기본정보, 소개, SNS, 포트폴리오 파일 업로드)

### 5. 고객센터
- FAQ (카테고리 필터 + 아코디언)
- 공지사항 (목록/상세)
- 1:1 문의 작성·목록·상세·답변
- 이용약관, 개인정보처리방침

### 6. 검색
- 강의명·강사명 통합 검색 (LIKE OR)
- 키워드 하이라이트 (`<mark>`)
- 검색 로그 기록 (`lc_search_log`)
- 결과 없음 상태 + 추천 검색어

### 7. 부가 기능
- 오픈채팅 입장 클릭 추적 (`lc_openchat_log`)
- 회사소개 (스크롤 sticky 섹션, 통계, 비전, 수강 후기)
- 반응형 레이아웃 (PC 기준 기획, 향후 모바일 대응 예정)
