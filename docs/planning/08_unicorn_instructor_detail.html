📋 화면 설명
1
GNB
클래스 리스트와 동일. 강사진 메뉴 active.
공통 컴포넌트
2
강사 사진
좌측 고정 영역. 사진 뒤에 V자 빨간 삼각형 장식 (타이탄클래스 동일 구조). 배경 제거된 인물 컷 권장. DB: lc_instructor.photo
사진 제공 필요 lc_instructor
3
강사명 + 카테고리
강사명(28px bold) + 카테고리 텍스트(회색). DB: lc_instructor.name, category
lc_instructor
4
소셜 아이콘
유튜브 / 인스타그램 / 틱톡 / 이메일 등. 해당 URL이 등록된 아이콘만 노출. 클릭 시 새 탭 열기. DB: lc_instructor.social_youtube, social_insta, social_tiktok, social_email
등록된 항목만 노출
5
강사소개
좌: "강사소개" 고정 레이블 / 우: 불릿(•) 리스트. DB: lc_instructor_intro 테이블 (instructor_idx FK, intro_text, sort_order). 개수 제한 없음.
lc_instructor_intro 내용 제공 필요
6
강사 경력
좌: "강사 경력" 고정 레이블 / 우: 불릿(•) 리스트. 강사소개와 동일한 레이아웃. 경력 미등록 시 "등록된 경력이 없습니다." 표시. DB: lc_instructor_career 테이블 (instructor_idx FK, career_text, sort_order)
lc_instructor_career 미등록 시 안내 문구
7
강의 목록 섹션
회색 구분선(border-top:8px solid #f5f5f5)으로 프로필 영역과 구분. "강의 목록" 타이틀. 해당 강사의 강의만 필터링.
lc_class WHERE instructor_idx
8
강의 카드
클래스 리스트와 동일한 카드 구조 (썸네일 16:9 + 뱃지 + 제목 2줄 + 강사명). 3열 그리드. 카드 클릭 시 강의 상세(03) 이동.
→ 강의 상세(03) 클래스 리스트 카드와 동일
📌 DB 연동 테이블
lc_instructor (기본 정보)
lc_instructor_intro (소개 항목)
lc_instructor_career (경력 항목)
lc_class (강의 목록, FK: instructor_idx)
