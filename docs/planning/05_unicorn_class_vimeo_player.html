아래 케이스를 클릭하면 위 플레이어에 해당 유효성 오버레이가 미리보기로 표시됩니다.
CASE 1 — 결제 완료 회원 (정상)
✅ 수강 가능
플레이어가 정상 노출되며 영상이 재생됩니다.
서버에서 결제 내역 확인 후 Vimeo video_id 반환
iframe embed 방식으로 삽입
CASE 2 — 비회원 접근
⚠ 로그인이 필요합니다.
플레이어 영역에 블러 오버레이 표시
URL 직접 접근 시에도 동일 처리
로그인 후 원래 페이지로 redirect
CASE 3 — 미구매 회원
✖ 구매 후 수강 가능합니다.
플레이어 영역에 블러 오버레이 + 결제하기 버튼
강의 상세 페이지로 이동 유도
video_id 비노출 (서버에서 미반환)
CASE 4 — 수강기간 만료
⚠ 수강기간이 만료되었습니다.
현재는 무제한 정책으로 해당 없음 (추후 기간제 도입 시 대비)
만료일 D-7부터 배너로 안내
재구매 또는 연장 기능 추후 결정
CASE 5 — 계정 정지 / 환불 완료
✖ 수강 권한이 없습니다.
환불 처리 또는 관리자 정지 시 접근 차단
플레이어 블러 오버레이 + 고객센터 안내
video_id 서버에서 미반환 처리
