📋 화면 설명
1
GNB — 화이트 헤더
클래스 목록과 동일한 화이트 테마. 강사소개 메뉴 active.
공통 컴포넌트
2
서브 배너
클래스 리스트와 완전 동일한 구조. 배경: #0d0d0d→#1a1a2e 그라디언트 + 좌측 빨간 오버레이. label "유니콘클래스의" + title "강사소개".
클래스 리스트와 동일
3
목록 헤더
"강사진" 타이틀 + 빨간색 총 인원 수. 탭/카테고리 필터 없음.
탭 없음 인원 수 DB 연동
4
강사 카드 그리드
4열 그리드. 한 페이지 최대 12명. DB: lc_instructor 테이블. sort_order 기준 정렬.
lc_instructor 최대 12명/페이지
5
강사 카드 구조
사진(height:240px 고정) + 이름 + 불릿 설명 1~2줄 + 소셜 아이콘(하단 고정). 호버 시: ① 카드 translateY(-8px) ② 사진 하단 빨간 그라디언트 오버레이 ③ 사진 위 소셜 아이콘 표시. CSS only.
사진 제공 필요 CSS hover only
6
강사 지원하기 CTA 섹션
강사 카드 그리드 하단 + 페이지네이션 위에 고정 노출. 다크 그라디언트 배경 (#1a1a2e→#0f2027) + 빨간 오버레이. "강사 지원하기 →" 버튼 클릭 시 /instructor/apply 이동.
12_unicorn_instructor_apply.html 연결
7
페이지네이션
클래스 리스트와 완전 동일한 구조 및 스타일. active: 빨간 배경(#c0392b). 12개 초과 시 다음 페이지 생성.
클래스 리스트와 동일 12개/페이지
📌 DB 연동 항목 (lc_instructor)
- instructor_idx (PK)
- name (강사명)
- photo (사진 경로)
- intro_1, intro_2 (불릿 설명)
- social_insta / youtube / blog / etc
- sort_order (노출 순서)
- is_active (노출 여부)
