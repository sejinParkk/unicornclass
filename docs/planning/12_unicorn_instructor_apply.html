2
서브 배너
강사 리스트와 동일한 다크 그라디언트. "강사 지원하기" 타이틀 + 간략 안내 문구.
강사 리스트와 동일 구조
3
기본 정보
이름·연락처·이메일·강의분야·강의경력 입력. 전항목 필수. DB: lc_instructor_apply.
필수 lc_instructor_apply
4
강사 소개
경력/자기소개(필수, 100자↑) + 강의계획(필수) + 희망형태(필수). 희망형태: 무료 웨비나/유료 VOD/혼합.
필수
5
SNS / 채널
인스타·유튜브·블로그·기타 URL 입력. 전체 선택 사항. 미입력 시에도 지원 가능.
선택
6
포트폴리오 첨부
파일 업로드(PDF/PPT/이미지, 최대 20MB, 3개) + 외부 링크 모두 지원. 둘 다 선택 사항. DB: lc_instructor_apply_file.
선택 파일 + 링크 지원
7
개인정보 동의
필수 동의. 미동의 시 제출 버튼 비활성. 수집 항목: 이름/연락처/이메일/강의정보. 보유 기간: 검토 완료 후 1년.
필수 동의
8
제출 버튼 + 완료 팝업
POST /api/instructor/apply 호출. 성공 시 완료 팝업 표시. 중복 제출 방지(제출 후 비활성). 관리자에서 지원자 목록·상태 관리.
POST /api/instructor/apply
