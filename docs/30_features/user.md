# 회원 기능 명세

## 1. 회원가입

### 플로우
1. `/register` 접근 → 회원가입 폼 렌더링
2. 아이디 중복 체크 (`POST /api/member/check-id`) → 실시간 AJAX
3. 폼 제출 → 서버 유효성 검사 → `lc_member` INSERT → 로그인 상태로 리다이렉트

### 입력 필드

| 필드 | 유효성 | 비고 |
|------|--------|------|
| 아이디 | 영문+숫자 4~20자, 중복 불가 | 실시간 중복 체크 |
| 비밀번호 | 8자 이상, 영문+숫자+특수문자 | eye 토글 (표시/숨김) |
| 비밀번호 확인 | 비밀번호와 일치 | |
| 이름 | 2~20자 필수 | |
| 이메일 | 이메일 형식 | |
| 휴대폰 | 숫자 11자리 | |

### 약관 동의

| 항목 | 필수 여부 |
|------|----------|
| 이용약관 동의 | 필수 |
| 개인정보처리방침 동의 | 필수 |
| 마케팅 정보 수신 동의 | 선택 |

- 전체 동의 체크박스로 일괄 동의 가능
- 각 약관 클릭 시 모달로 내용 표시
- 필수 약관 미동의 시 제출 불가

### 보안
- 비밀번호: `password_hash()` (PASSWORD_BCRYPT)
- CSRF 토큰 필수

## 2. 로그인

### 이메일/ID 로그인
- URL: `POST /login`
- 입력: 아이디 or 이메일 + 비밀번호
- `password_verify()` 검증
- 성공: 세션 발급, 이전 페이지 또는 메인으로 이동
- 실패 케이스:
  - 존재하지 않는 계정 → 에러 메시지
  - 비밀번호 불일치 → 에러 메시지
  - 탈퇴 계정 → 팝업 ("탈퇴한 계정입니다")
  - 휴면 계정 → 본인 인증 후 활성화 유도

### 소셜 로그인 (카카오)
1. `/auth/kakao` → 카카오 인증 서버로 리다이렉트
2. 콜백 `/auth/kakao/callback` → 인가코드 수신
3. Access Token 발급 → 사용자 정보 조회
4. `lc_member`에서 `social_provider=kakao AND social_id=?` 조회
   - 존재: 로그인 처리
   - 미존재: 이메일 없는 경우 추가 정보 입력 폼 → 신규 회원 생성

### 소셜 로그인 (네이버)
- 카카오와 동일 흐름, `social_provider=naver`

## 3. 아이디 찾기

- URL: `GET/POST /find-id`
- 입력: 휴대폰 번호 → SMS 인증번호 발송 (`POST /api/member/send-sms`)
- 인증번호 입력 → `POST /api/member/verify-sms`
- 인증 성공 → 아이디 마스킹 표시 (예: `hong***`)
- 인증 타이머: 3분

---

## 4. 비밀번호 재설정

- URL: `GET/POST /find-password`
- 입력: 가입 이메일 → 재설정 링크 발송
- 링크 클릭 → 새 비밀번호 입력 폼
- 비밀번호 저장 후 로그인 페이지로 이동
- 토큰 유효시간: 30분

## 5. 마이페이지

### 5-1. 나의 강의 (`/mypage/my-class`)

**강의 카드 상태 구분:**

| 상태 | 조건 | UI |
|------|------|----|
| 수강중 | `expire_at > NOW()` 또는 무료강의 | 강의 보기 + 오픈채팅 버튼 활성 |
| 만료·연장 가능 | `expire_at <= NOW()` AND `lc_class.sale_end_at > NOW()` | 기간 연장 버튼 (클래스 상세로 이동) |
| 만료·연장 불가 | `expire_at <= NOW()` AND `lc_class.sale_end_at <= NOW()` | "강의 판매 종료" 안내, 연장 불가 버튼 |
| 수료 | `lc_progress.rate >= 80` | 수료증 발급 버튼 표시 |

**탭 필터:** 전체 / 무료강의 / 프리미엄강의
**페이지네이션:** 10개/페이지
**API:** `GET /api/mypage/classes?page=1&limit=10&type=all`

### 5-2. 찜목록 (`/mypage/wishlist`)
- 찜한 강의 그리드 (3열)
- 찜 해제 버튼 → `POST /api/classes/{idx}/wish` 토글
- 빈 상태: "찜한 강의가 없습니다" 안내

### 5-3. 결제내역 (`/mypage/orders`)
- 결제일, 강의명, 금액, 상태 목록
- 상세 조회 → 영수증 출력
- 환불 신청 버튼 (환불 가능 조건일 때만 노출)

### 5-4. 1:1 문의 (`/mypage/qna`)
- 문의 목록 (제목, 접수일, 처리 상태)
- 문의 작성: 카테고리 선택 + 제목 + 내용
- 상세: 질문 + 관리자 답변

### 5-5. 후기 (`/mypage/reviews`)
- 수강 완료 강의 기준 후기 작성 가능
- 별점 (1~5) + 텍스트
- 수정/삭제 가능 (본인만)

### 5-6. 정보수정 (`/mypage/profile`)

| 필드 | 수정 가능 여부 |
|------|--------------|
| 이름 | 가능 |
| 이메일 | 가능 (중복 체크) |
| 휴대폰 | 가능 |
| 비밀번호 | 현재 비밀번호 확인 후 변경 |
| 프로필 사진 | 업로드 가능 |
| 아이디 | 변경 불가 |

### 5-7. 로그아웃
- `POST /logout` → 세션 완전 파기 → 메인 이동

### 5-8. 회원탈퇴 (`/mypage/withdraw`)
- 비밀번호 확인 후 진행
- `mb_status = 'withdrawn'` 처리 (물리 삭제 아님)
- 탈퇴 후 동일 이메일/소셜로 재가입 불가 기간: 30일

---

## 6. 수료증 발급

- 조건: `lc_progress.rate >= 80`
- 마이페이지 > 나의 강의 > 수료증 발급 버튼
- PDF 다운로드 형태
- 발급 이력 저장 (`lc_certificate` 테이블 추가 가능)
